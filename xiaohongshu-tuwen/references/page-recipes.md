# Page Recipes

## 目录

- 封面页
- KPI 数据页
- 双列对比页
- 趋势图页
- 排名 / 横条页
- 解释 / 启示页
- 结论 / 行动页
- 推荐轮播组合

下面是最适合复刻参考图风格的页面模板。默认不必全用，按主题选 5 到 7 页即可。

## 1. 封面页

最适配模板：

- `signal-alert`
- `premium-brief`
- `data-newsroom`

适用：

- 提出一个强判断
- 抛出一个问题
- 给出一个反常识结论

结构：

- 顶部小角标或账号标识
- 2 到 3 行主标题
- 关键词单独高亮
- 一个短副标题或导语
- 可选淡化背景字母 / 几何装饰

要求：

- 标题最好能被切成两段形成节奏
- 不要把封面写成摘要
- 只需要让人想继续滑

## 2. KPI 数据页

最适配模板：

- `data-newsroom`
- `premium-brief`

适用：

- 展示总量、增速、比例、同比

结构：

- 左右两个或三个数据卡
- 每张卡只承载一个核心数字
- 下方用短条形进度或小注解释来源

要求：

- 大数字必须比其他元素更先被看到
- 每张卡的说明不要超过两行

## 3. 双列对比页

最适配模板：

- `benchmark-board`
- `playbook-cards`

适用：

- 两个产品、两类人群、两种路径、两个时代的对比

结构：

- 左右双列
- 左侧列出对比维度
- 两边卡片样式统一，但主推一侧可以强化颜色

适合维度：

- 核心理念
- 成本
- 增长速度
- 使用门槛
- 记忆方式
- 安全性

要求：

- 维度不超过 7 个
- 每格内容尽量压缩成短标签，而不是句子

## 4. 趋势图页

最适配模板：

- `data-newsroom`
- `signal-alert`

适用：

- 时间序列变化
- 周期走势
- 两类群体的变化对比

结构：

- 上部为标题和一句解释
- 中部放折线图
- 下部补充 1 到 2 个解释卡片

要求：

- 折线颜色不能过多
- 坐标和注释弱化，主线条和节点强化
- 只标最关键的拐点

## 5. 排名 / 横条页

最适配模板：

- `benchmark-board`
- `data-newsroom`

适用：

- 公司排名
- 裁员规模
- 市占率排序
- 功能覆盖对比

结构：

- 左侧标签
- 中间深色条或浅色条表现量级
- 右侧补一条时间或备注

要求：

- 排序必须清晰
- 文本长度要统一，避免视觉参差太大

## 6. 解释 / 启示页

最适配模板：

- `explainer-notebook`
- `playbook-cards`
- `signal-alert`

适用：

- 回答“这意味着什么”
- 给出用户最关心的后果

结构：

- 左右双卡，或上下一主一次
- 每张卡聚焦一个启示
- 可配简化图示、占比环图、结论标签

要求：

- 这一页要从“信息”进入“解读”
- 不要重新复述前几页的数据

## 7. 结论 / 行动页

最适配模板：

- `playbook-cards`
- `signal-alert`
- `premium-brief`

适用：

- 总结判断
- 给出建议
- 为评论区互动埋钩子

结构：

- 1 句主结论
- 3 个以内的行动建议或判断分支
- 页尾可加“你怎么看”类互动句

要求：

- 结论要能独立成立
- 不要写空洞鸡汤

## 推荐轮播组合

### 组合 A：观点判断型

1. 封面页
2. 核心数据页
3. 对比页
4. 趋势页
5. 启示页
6. 结论页

### 组合 B：行业盘点型

1. 封面页
2. 总览页
3. 排名页
4. 趋势页
5. 原因页
6. 机会页

### 组合 C：产品拆解型

1. 封面页
2. 产品定位页
3. 双列对比页
4. 使用门槛页
5. 成本 / 效率页
6. 谁更适合谁页
